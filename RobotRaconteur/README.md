# RobotRaconteur turtle module example
![](images/turtle.gif)

## Prerequisite:
### tkthread:
`pip install tkthread`
### tcl-thread:
#### Ubuntu:
`sudo apt-get install tcl-thread`
#### Windows:
Move folder *thread2.8.4* to *C:\<YourPythonInstallationDir>\tcl\tcl8.6*
### pygame:
`pip install pygame`
